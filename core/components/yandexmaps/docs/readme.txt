**Yandex Maps for MODX Revo**

TV-тип tvYandexMap для выбора координат на карте в админке + сниппет [[YandexMap]].
Системные настройки:

- yandexmaps.api_key
- yandexmaps.default_center
- yandexmaps.default_zoom

После установки:

- Укажите API ключ Яндекс Карт в Система → Настройки → yandexmaps.api_key.
- Создайте TV с типом ввода tvYandexMap и привяжите к нужным шаблонам.
- В шаблоне используйте [[YandexMap? &tv=`coords`]].
